# Emergency-department-demand
## Dataset

This project uses the Hospital Triage and Patient History dataset:
https://www.kaggle.com/datasets/maalona/hospital-triage-and-patient-history-data

Due to GitHub file size limits, the dataset CSV files are not included in this
repository. Download the dataset and place it as `triage_data.csv` in the
project root, then run phase1.py through phase6.py in order.
